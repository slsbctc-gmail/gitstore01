# 溪海钓手

一个 Windows 图形桌面钓鱼游戏原型。程序使用 WinForms 构建，最终产物是 `dist\FishingGame.exe`，不是网页版。

## 运行

构建后双击运行：

```powershell
.\dist\FishingGame.exe
```

首次启动会在可执行文件旁边创建 `fishing-save.json` 存档。删除该文件可以重新开始。

## 构建

本项目使用 Windows 自带的 .NET Framework C# 编译器，不需要安装 .NET SDK。

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1
```

只运行核心测试：

```powershell
powershell -ExecutionPolicy Bypass -File .\build.ps1 -TestOnly
```

## 当前内容

- 12 个钓鱼场景，难度从晴溪浅滩到龙潮禁海递增。
- 每个场景 20-100 种鱼，前期图鉴更轻，后期逐步增加难度；全游戏鱼种名称不重复。
- 每个场景 1-2 条隐藏鱼，基础出现概率很低。
- 每个场景有隐藏非鱼类物品，例如手表、宝石、手机等，可以出售换金币。
- 完成当前场景全部非隐藏鱼图鉴后，自动解锁下一个场景。
- 24 种鱼竿，用金币购买，力量、控制、幸运和稳定性逐步提高。
- 幸运值会缩短等待倾向、提高稀有鱼权重，并提高隐藏鱼概率但有上限。
- 每种鱼有不同控线特性，安全区、拉线效果、放线效果和挣扎幅度都会变化。
- 鱼按重量分为偏小、一般、极品；偏小鱼会自动放生，一般鱼按重量卖，极品鱼有 2-10 倍高价。
- 鱼可以出售，也可以放入鱼缸收藏。
- 鱼缸容量从普通鱼缸 5 条升级到城市水族馆 1000 条。
- 升级到观赏鱼廊后，可以每天收取一次门票金币。
- 每日签到、售卖鱼、首次图鉴奖励和门票都是金币来源。

## 操作

- 点击左侧场景切换钓点。
- 点击“抛竿”开始钓鱼。
- 鱼咬钩后用 `Space` 或“拉线”提高张力。
- 用 `S`、方向下键或“放线”降低张力。
- 拉线和放线是圆形按钮；让张力指针保持在安全区内，进度满后捕获成功。
- 背包里的鱼可以出售，也可以放入鱼缸。
- 鱼缸里的鱼不会被一键出售，可以移回背包后再卖。

## 验证

核心测试覆盖：

- 场景数量。
- 每场景鱼种数量范围。
- 每场景隐藏鱼数量。
- 每场景隐藏物品。
- 全局鱼名唯一。
- 场景鱼种数量从轻到重递增。
- 鱼竿数量。
- 鱼缸最小/最大容量。
- 观赏鱼廊门票解锁。
- 幸运值提高隐藏鱼概率。
- 鱼和鱼竿会改变张力安全区。
- 鱼的拉线和放线特性不同。
- 重量等级控制放生和售价。
- 非隐藏鱼图鉴解锁下一场景。
- 鱼缸容量限制。
- 门票每天只能收取一次。
- JSON 存档回读。
- UI 烟测会把所有场景绘制到内存图像，防止 WinForms 红叉和绘图异常回归。
